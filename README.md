# Random

A simple website for when you need some randomness. Whether you need a random number, string, UUID, or just want to roll the dice, Random has got you covered.

## Features

- **Select**: Randomly select one from a variety of options
- **Numbers**: Generate random numbers
- **Strings**: Generate random strings
- **UUID**: Generate random UUIDs
- **Roll**: Roll the dice
